# IT-140-zombie-text-adventure

**Overview**
This project is a text-based adventure game developed in Python as part of IT-140. The game allows the player to navigate rooms, collect items, and attempt to escape during a zombie apocalypse.

**Gamplay Features**
- Room navigation using directional commands
- Inventory system implemented through Python list
- Dictionary based room mapping
- Win/loss condition based on items collected

**Technical Structure**

The program uses:
 - A dictionary to represent room adjacency relationships
 - A list that tracks player inventory
 - A procedural while-loop as the game engine
 - Conditional logic to determine win/loss state
